# 印尼語語音 / LLM 金鑰申請步驟（給老師）

> 模擬器自測**完全不用金鑰**。以下是要「接真語音 / 真 LLM」時才需要。
> 重點：語音層是抽象介面，**先用最簡單的，之後可免費換更自然的印尼語語音，程式碼不用改**。

---

## 🥇 最推薦（最省事）：OpenAI 一個帳號搞定全部
一把金鑰同時涵蓋 **語音辨識(Whisper STT) + 語音合成(TTS) + LLM(GPT)**，**免信用卡**（註冊送 $5 額度），無月費、用多少算多少。

**步驟：**
1. 開 **https://platform.openai.com/signup** ，用 Google 帳號或 email 註冊（可能需手機簡訊驗證）。
2. 登入後，右上角頭像 → **View API keys**（或直接開 https://platform.openai.com/api-keys ）。
3. 點 **Create new secret key** → 取個名字（如 `kebbi`）→ 建立。
4. **金鑰只顯示一次**，長得像 `sk-...`，馬上複製存好。
5. （$5 用完才需加值；加值才需信用卡，最低 $5。一場比賽用量很省。）

> 印尼語：Whisper 辨識印尼語沒問題；TTS 也會說印尼語，只是腔調比原生語音略「外國」一點——對 demo 夠用，要更自然再走下面的升級。

---

## 🥈 升級印尼語發音（免信用卡）：ElevenLabs 配 OpenAI
語言課的發音是教學內容，想要**最自然的印尼語**又**不想綁信用卡**，就用這組：
- **TTS（機器人講印尼語）→ ElevenLabs**（免卡、1 萬字/月、印尼語很自然）
- **STT（聽學生）+ LLM → OpenAI**（同上）

**ElevenLabs 步驟：**
1. 開 **https://elevenlabs.io/app/sign-up** ，用 Google 註冊（免信用卡）。
2. 登入 → 左下角頭像 → **API Keys**（或 Profile）→ 複製金鑰。
3. 免費方案每月 1 萬字、永久重置。

---

## 🏅【✅ 已選定路線】最標準的原生印尼語：Azure 一般版
Azure 有原生印尼語神經語音 `id-ID-GadisNeural`（女）/`id-ID-ArdiNeural`（男），最標準。
> 註：學校信箱版（Azure for Students）`grad.ntue.edu.tw` 驗證失敗（台灣 .edu.tw 子網域常無法自動驗證），故改走一般版。

**Step 1 — 註冊免費帳號**（要綁卡驗證身分，**不會扣款**）：https://azure.microsoft.com/free
- F0 語音層「永久免費」，不會自動扣款；30 天 $200 試用額度到期只會停用、不轉收費。

**Step 2 — 建 Speech 資源：**
1. Azure Portal → 搜尋 **Speech** → **Speech services** → **Create**。
2. Resource group 新建（如 `kebbi`）；Region 選 **Southeast Asia**；**Pricing tier 選 `Free F0`**。
3. Review + Create → Create，等部署完成。

**Step 3 — 拿金鑰：**
1. 進資源 → 左側 **Keys and Endpoint** → 複製 **KEY 1** 與 **Location/Region**。
2. ⚠️ **常見坑**：region env 變數要填**小寫、無空格**代號 → Southeast Asia 寫成 **`southeastasia`**。

## 🔐 金鑰拿到後怎麼交給我（重要：別貼在對話裡）
金鑰等於密碼，貼進對話會留紀錄。請在這台電腦設環境變數，我就讀得到：

```bash
# 路線一：全 OpenAI（一把金鑰當語音也當 LLM）
export KEBBI_SPEECH_KEY="sk-..."        # 同一把 OpenAI 金鑰
export KEBBI_LLM_KEY="sk-..."           # 同一把 OpenAI 金鑰

# 路線二：ElevenLabs(TTS) + OpenAI(STT+LLM)
export KEBBI_TTS_KEY="elevenlabs金鑰"
export KEBBI_SPEECH_KEY="sk-..."        # OpenAI（STT）
export KEBBI_LLM_KEY="sk-..."           # OpenAI（LLM）

# 路線三：Azure
export KEBBI_SPEECH_KEY="azure金鑰"
export KEBBI_SPEECH_REGION="southeastasia"
export KEBBI_LLM_KEY="sk-...或Claude金鑰"
```

設好後跟我說「設好了，走路線X」，我就把對應的雲端後端接上、加自測，讓模擬器用**真印尼語語音 + 真 LLM**跑起來（這階段還不用實體機器人）。
