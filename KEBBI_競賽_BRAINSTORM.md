# Kebbi 凱比機器人競賽 — 培訓與選題 Brainstorm

> 對象競賽：**STEM 人文跨域與 AI 應用競賽 — Kebbi 凱比機器人創新教案與應用競賽**
> 主辦：東吳大學巨量資料管理學院 ｜ 兩大主題：**教案** × **生活應用**
> 本文件用途：替不同程度學生培訓參賽，從 Kebbi 的硬體特性出發做功能組合，發想可奪獎的題目。

---

## 一、競賽快照（先對齊規則，再發想）

| 項目 | 重點 |
|---|---|
| 競賽主題 | 以凱比機器人為核心，設計**創新教學或生活應用**方案，結合人文精神、STEM 與「以人為本」需求，主題不限 |
| 兩大方向 | **教案組**（正式/非正式教育情境）＋**生活應用**（日常情境） |
| 組別 | 一般教案組（含國中小、高中職、大專師生）、大專跨域組（須跨系所） |
| 隊伍 | 每隊 ≤ 8 人，學生須有指導老師；組長須中華民國國籍；未滿 18 歲需家長同意書 |
| **初賽評分** | 應用性 **35**／創意性 **35**／挑戰性 **30**（系統概述 ≤ 3 頁，教案格式） |
| **決賽評分** | 初賽成績 **30**／完整性（含 **Kebbi 整體成果**）**40**／簡報與展演 **30** |
| 決賽作品 | **直式影片 720×1280、3–5 分鐘、≤200MB（mp4）** ＋ 簡報 ＋ **現場實體展演約 6 分鐘** |
| 時程（沿用 114 年節奏，115 年以官網為準） | 6–8 月工作坊 → 8 月中初賽交件 → 8 月底公告入圍 → 9 月底繳影片＋決賽實體展演 |
| 獎項 | 創新卓越 15,000／創新優等 8,000／展演卓越 6,000／佳作 3,000／最佳人氣 |

### 從評分反推的選題戰略
1. **「應用性」是最大分項（初賽 35 分）**：題目一定要打中一個**真實、具體、有對象**的需求（誰、什麼情境、解決什麼痛點），不要做「炫技但沒人用」。
2. **決賽「完整性 40 分」明寫含「Kebbi 機器人整體成果」**：作品必須**真的在 Kebbi 上跑起來**，不能只是 PPT 概念。
3. **「展演 30 分」＋現場 6 分鐘實體展演**：會動、會轉頭、會比手勢的橋段最吃香——這正是 Kebbi 勝過平板/聊天機器人的地方，要留「演出時刻」。
4. **115 年評審專長透露的偏好**：李忠謀（影像處理／電腦視覺）、陳玫蓉（英語語言）、蔡智孝（數學與 AI 教育）、林孝綱（機器人教學）。➜ **善用攝影機視覺辨識、英語/語言學習、數學與 AI 素養**的題目較易引起評審共鳴。

## 二、Kebbi 功能 → 能力地圖

把硬體翻譯成「教學/生活上能做什麼」，並標出**只有機器人做得到、平板做不到**的差異點（這是拿創意分與展演分的關鍵）。

| Kebbi 功能 | 能做的事 | 差異化亮點（vs. 平板/手機 App） |
|---|---|---|
| 觸控螢幕 | 出題、選項、圖卡、表情、計分板、無障礙大字介面 | 表情會回應情緒 → 有「臉」的陪伴感 |
| 方向性麥克風 | 聽指令、辨識**聲音來自哪個方向** | **聲源定位**：知道「誰在說話、在哪邊」 |
| 喇叭 + TTS | 念題、說明、唱歌、旁白、雙語發音 | 即時把任何文字變語音，可做主持人/說書人 |
| ASR（語音辨識） | 聽懂口說、口說評分、語音問答 | 免打字，低年級/長者也能用 |
| 攝影機 | 物件/人臉/表情/手勢/姿態辨識、拍照記錄 | **會看**：看你做對沒、看桌上有什麼、認得你是誰 |
| 頭、手關節 | 比手勢、點頭、肢體語言、帶動唱、示範動作 | **有身體**：能「演」、能示範、能陪跳 |
| 前進/後退/旋轉移動 | 走位、轉向說話者、巡邏、導覽、闖關 | **會移動**：能來找你、帶你走、在空間中互動 |
| 網路 | 連雲端 AI/LLM、查資料、IoT、跨裝置 | 接 GenAI 與外部資料 → 內容無限、會即時更新 |

> **一句話記住產品特色**：Kebbi ＝「**會看（攝影機）＋會聽方向（指向麥）＋會動會比（關節＋移動）＋會講（TTS/ASR）＋連網（雲端 AI）**」的**有身體的 AI 夥伴**。題目越用到「身體 + 空間 + 感知」，越能贏過「螢幕上的聊天機器人」。

## 三、招牌功能組合（發揮產品特色的七個「殺手鐧」）

這些組合是發想引擎——任一題目都建議至少套用其中 2 個，讓 Kebbi 的特色被看見。

| # | 組合 | 解鎖的體驗 | 適合 |
|---|---|---|---|
| **C1 看了就回饋** | 攝影機 + 雲端視覺 + TTS/螢幕 | 辨識物件/動作/表情 → 即時口語＋表情回饋（垃圾分類、植物辨識、握筆姿勢、運動矯正） | 全年齡，視覺題首選 |
| **C2 聽聲辨位** | 方向性麥 + 旋轉/移動 + 攝影機 | 誰說話就**轉頭看誰**、走向發言者；可做點名、主持、互動劇 | 國中以上，展演加分強 |
| **C3 蘇格拉底對話** | ASR + 雲端 LLM + TTS | 不直接給答案，用追問引導思考；口說練習、思辨、語言對話 | 高中/大專，挑戰性高 |
| **C4 會演的老師** | 頭手關節 + 移動 + TTS/喇叭 | 帶動唱、戲劇演出、肢體示範、節奏律動、手語/動作教學 | 國小、語言、藝術 |
| **C5 多模態無障礙** | 觸控螢幕 + ASR + TTS + 表情 | 圖＋語音＋語音輸入，照顧不識字/視障/長者/特教 | 生活應用、特教 |
| **C6 移動關卡/導覽** | 移動 + 攝影機 + 觸控 + 網路 | 闖關尋寶、校園/展館導覽、巡檢提醒 | 國小活動、生活應用 |
| **C7 姿態教練** | 攝影機(姿態辨識) + 關節示範 + 語音 | 看你的動作→示範正確版→口頭糾正（體育、復健、書法、樂器） | 高中/大專，挑戰性高 |

## 四、選題策略：避開三個常見陷阱

- ❌ **「平板就能做」陷阱**：純問答 App、純朗讀、純測驗——沒用到身體/移動/聲源/視覺，創意分難拿。➜ 至少加一個「會動」或「會看」的環節。
- ❌ **「炫技無對象」陷阱**：功能很多但說不出「給誰、解決什麼」。➜ 先寫一句**使用者故事**：「（某人）在（某情境）想要（某結果），但（某痛點）。」
- ❌ **「PPT 作品」陷阱**：決賽要實體展演，做不出能跑的雛形＝完整性 40 分大失血。➜ 培訓時就以「能 demo 的最小版本」為目標。

## 六、各級別詳細點子

> 每個點子的格式：**需求/對象 → Kebbi 功能組合 → 怎麼跑（流程）→ 對應評分亮點 → 挑戰度（★）與延伸**。

### ▍小五升六（約 10–11 歲）
發展特性：具象思維、愛遊戲與競賽、注意力短、適合**積木式視覺程式**。Kebbi 當「會動的玩伴」最有共鳴。

**E1｜凱比闖關・數學王國**（教案）
- 需求：低年段數學練習枯燥，缺乏即時回饋與成就感。
- 組合：**C6 移動關卡 + C4 會演 + 觸控/TTS**。
- 流程：Kebbi 在教室地面移動到「關卡站」→ 出題（螢幕＋語音）→ 學生口答或觸控 → 答對則**轉圈＋舉手歡呼**、答錯則引導再試 → 集滿關卡解鎖「寶藏」動畫。
- 評分亮點：應用性（課堂可直接用）＋展演（會走會演，現場超吸睛）。
- 挑戰度：★★☆　延伸：題庫可換成國語成語、英語單字。

**E2｜小小自然觀察家**（教案）
- 需求：自然課植物/昆蟲辨識，課本圖片靜態、學生記不住。
- 組合：**C1 看了就回饋 + 螢幕圖卡**。
- 流程：學生把葉片/昆蟲圖卡（或實物）拿到攝影機前 → Kebbi 辨識並用語音講特徵、棲地、小故事 → 出 1 題小測驗鞏固。
- 評分亮點：用到攝影機視覺（評審李忠謀專長）＋戶外/教室皆可。
- 挑戰度：★★☆　延伸：做成「校園植物地圖」蒐集冊。

**L1｜校園導覽小嚮導**（生活應用）
- 需求：新生、家長、來賓不熟悉校園。
- 組合：**C6 移動導覽 + 觸控選單 + TTS**。
- 流程：在觸控螢幕選目的地（圖書館/保健室/廁所）→ Kebbi 帶路前進並沿途介紹 → 抵達比手勢說明。
- 評分亮點：移動是核心亮點，生活情境清楚。
- 挑戰度：★★☆（需處理路線/避障，可先用定點地標版）。

**L2｜整理小達人**（生活應用）
- 需求：低年級教室物品亂放、不會收納。
- 組合：**C1 視覺辨識 + C5 多模態提醒**。
- 流程：攝影機看桌面/櫃子 → 指出「鉛筆盒沒歸位」並用圖＋語音提示 → 完成後給星星貼紙鼓勵。
- 挑戰度：★★☆。

### ▍國二升國三（約 13–14 歲）
發展特性：抽象思維萌芽、面臨**會考**、開始做實驗、在意同儕。可寫**進階積木或入門文字程式**。

**E3｜理化實驗安全＆步驟教練**（教案）
- 需求：實驗步驟多、易出錯、有安全風險，老師難一對一盯。
- 組合：**C1 視覺辨識 + C7 姿態/動作檢查 + TTS 旁白**。
- 流程：Kebbi 念出步驟並示範 → 攝影機檢查器材擺放/動作（如有沒有戴護目鏡、酒精燈是否正確點燃）→ 偵測到危險立即出聲提醒。
- 評分亮點：應用性極強（安全是真痛點）＋挑戰性（視覺判讀）。
- 挑戰度：★★★。

**E4｜英語口說擂台**（教案）
- 需求：學生不敢開口說英語、缺練習對象。
- 組合：**C3 對話 + C2 聽聲辨位 + ASR 口說評分**。
- 流程：Kebbi 出情境題（點餐、問路）→ 學生口說 → ASR 評發音/流暢度給分 → **轉頭看向發言同學**做問答 → 排行榜激勵。
- 評分亮點：英語語言（評審陳玫蓉專長）＋口說評分有技術挑戰。
- 挑戰度：★★★。

**L3｜班級事務管家**（生活應用）
- 需求：點名、值日生輪值、收作業、午休秩序耗老師時間。
- 組合：**C2 聽聲辨位 + C5 多模態 + 攝影機（人臉/舉手）**。
- 流程：早上人臉點名 → 公告今日值日生與行程 → 午休偵測噪音方向並溫和提醒 → 統計繳交狀況。
- 挑戰度：★★★（人臉辨識需注意隱私，可改用座號/QR）。

**L4｜讀書計畫陪伴教練**（生活應用）
- 需求：會考壓力大、自律難、缺陪伴。
- 組合：**C3 對話 + C4 會演（番茄鐘律動）+ TTS**。
- 流程：對話設定每日目標 → 番茄鐘專注計時，休息時帶 1 分鐘伸展律動 → 完成打氣、未達標溫柔追問卡關原因。
- 挑戰度：★★☆。

### ▍國三升高一（約 14–15 歲）
發展特性：**銜接與轉換期**、開始生涯探索、自主學習、認同感重要。題目可帶「探索自我、適應新環境」的人文味（呼應競賽「以人為本」精神）。

**E5｜生涯探索對話夥伴**（教案）
- 需求：選高中/類組/技職很迷惘，缺乏結構化自我探索。
- 組合：**C3 蘇格拉底對話 + C2 看著你說話 + 螢幕視覺化**。
- 流程：透過興趣/性格問答（類 Holland 碼）對話 → Kebbi 追問而非直接下結論 → 螢幕呈現興趣雷達圖與可能方向 → 提供延伸探索任務。
- 評分亮點：人文關懷 + 對話設計有挑戰性。
- 挑戰度：★★★。

**E6｜跨科 STEAM 專題引導員**（教案）
- 需求：專題式學習（PBL）學生不知如何發想與聚焦。
- 組合：**C3 對話引導 + C1 視覺（看學生的草圖/原型）+ 網路查資料**。
- 流程：用提問引導定義問題 → 看學生手繪原型給回饋 → 連網找參考資料與案例 → 產出專題計畫骨架。
- 挑戰度：★★★。

**L5｜新生適應導覽＆破冰主持**（生活應用）
- 需求：剛升上新學校，人際陌生、環境不熟、易焦慮。
- 組合：**C2 聽聲辨位主持 + C6 移動導覽 + C4 帶動破冰遊戲**。
- 流程：開學日 Kebbi 帶隊導覽校園 → 在班上主持破冰小遊戲，**轉向回答的同學**互動 → 帶一段團康律動暖場。
- 評分亮點：展演非常強（移動＋主持＋律動），情境溫暖。
- 挑戰度：★★★。

**L6｜情緒支持與紓壓夥伴**（生活應用）
- 需求：課業壓力、情緒無處說。
- 組合：**C5 多模態 + C4 引導呼吸/伸展 + C3 同理對話**。
- 流程：表情/語氣偵測情緒 → 同理回應、帶引導式呼吸與肩頸伸展 → 嚴重時提示求助管道（非診斷，重在陪伴與轉介）。
- 挑戰度：★★★（務必標明「非醫療、僅陪伴與轉介」）。

### ▍高一升高二（約 15–16 歲）
發展特性：學科加深、能寫**較完整程式（可串接 API/雲端 AI）**、適合做有資料、有系統感的作品，已能挑戰接近大專水準。

**E7｜數學/物理概念可視化助教**（教案）
- 需求：函數、向量、力學等抽象概念難想像。
- 組合：**C1 視覺（辨識手寫式子/實物）+ C3 講解對話 + 螢幕動態圖**。
- 流程：學生寫下函數或擺出實驗（如斜面）→ 攝影機辨識 → Kebbi 在螢幕畫出對應圖形/受力圖並口頭講解 → 追問檢核理解。
- 評分亮點：數學與 AI 教育（評審蔡智孝專長）＋視覺辨識挑戰。
- 挑戰度：★★★★。

**E8｜雙語簡報與口說評分教練**（教案）
- 需求：學生簡報時眼神、語速、肢體、發音都缺回饋。
- 組合：**C7 姿態辨識 + C3 ASR 評分 + C2 看著練習者**。
- 流程：學生對 Kebbi 練習簡報 → 攝影機看姿態/眼神接觸、ASR 評語速與發音、偵測冗詞 → 產出回饋報告與改善建議。
- 挑戰度：★★★★。

**L7｜校園無障礙／長者陪伴助理**（生活應用）
- 需求：行動不便者、長者使用 3C 與獲取資訊有障礙。
- 組合：**C5 多模態無障礙 + C2 循聲找人 + 網路（天氣/用藥/行程）**。
- 流程：純語音操作查詢資訊 → 聽到呼喚循聲走近 → 大字＋語音播報提醒（吃藥、回診、天氣）。
- 挑戰度：★★★★。

**L8｜社區永續・智慧資源回收站**（生活應用）
- 需求：垃圾分類常分錯、回收教育成效低。
- 組合：**C1 視覺分類辨識 + C6 移動/互動 + 數據統計**。
- 流程：把回收物拿到攝影機前 → 辨識並指示正確分類、講解理由 → 累積班級/社區減碳數據做成排行榜。
- 評分亮點：SDGs/永續主題討喜，視覺辨識有挑戰。
- 挑戰度：★★★★。

### ▍高三升大學（約 17–18 歲，可衝大專級完整度）
發展特性：能做**完整系統整合**、串接外部資料/LLM、處理多模態與真實場域問題。建議鎖定「**社會影響力 + 技術完整度**」雙高的旗艦題，直攻創新卓越獎。

**E9｜適性學習診斷家教系統**（教案）
- 需求：班級內程度落差大，老師無法一對一補救。
- 組合：**C3 對話家教 + C1 看作答過程 + C7 看書寫/解題姿態 + 網路（題庫/LLM）**。
- 流程：診斷測驗找出弱點 → 針對弱點出題並用蘇格拉底式提問引導 → 攝影機看解題步驟找錯誤點 → 動態調整難度，產出學習歷程報告給師生。
- 評分亮點：應用性＋完整性＋挑戰性全包，最接近「產品級」。
- 挑戰度：★★★★★。

**E10｜多語導覽／手語雙向溝通夥伴**（教案／可跨生活應用）
- 需求：聽障者與一般人溝通障礙；多語族群資訊落差。
- 組合：**C1 手勢/手語辨識 + C4 關節做手語動作 + C3 多語 TTS/ASR 翻譯**。
- 流程：看懂使用者手語 → 轉成語音/文字；反向把語音轉成 Kebbi 的**肢體手語示範**與字幕 → 達成雙向溝通。
- 評分亮點：人文關懷頂級、**身體（關節做手語）是平板絕對做不到的**、技術極具挑戰。
- 挑戰度：★★★★★（可先做有限詞彙的概念驗證版）。

**L11｜獨居長者健康關懷站**（生活應用）
- 需求：獨居長者孤獨、用藥與異常無人即時察覺。
- 組合：**C1 視覺（跌倒/表情）+ C2 循聲互動 + C5 大字語音 + 網路（通報家人）**。
- 流程：每日問候與用藥提醒 → 偵測異常（跌倒、長時間無回應）→ 連網通報家屬/里長 → 平時陪聊解悶。
- 評分亮點：高齡社會剛需、系統完整、串接通報具挑戰。
- 挑戰度：★★★★★。

**L12｜校園心理健康初篩與陪伴夥伴**（生活應用）
- 需求：青少年情緒困擾普遍，輔導人力不足、學生不願主動求助。
- 組合：**C3 同理對話 + C5 多模態 + 表情/語氣分析 + 網路（轉介資源）**。
- 流程：以匿名、友善對話了解狀態 → 用量表化問題評估 → **明確界線：不做診斷**，提供自助資源與轉介專業輔導 → 統計（去識別化）供輔導室掌握趨勢。
- 挑戰度：★★★★★（倫理/隱私設計是評分與展演的重點，要主動講清楚）。

## 七、跨級別旗艦題（衝大獎用，建議高三/大專隊）

把多個招牌組合疊起來、有真實場域、能講出社會價值的題目，最容易在「應用性＋完整性＋展演」三個大分項同時拿高分：

- **無障礙溝通夥伴（E10 手語雙向）** — 賣點：Kebbi 用「身體」做手語，獨一無二。
- **適性家教系統（E9）** — 賣點：最像可商品化的教育產品，完整度高。
- **長者關懷站（L11）** — 賣點：對準高齡社會剛需，情感打動評審。

> 旗艦題的風險是「做不完」。培訓策略：先定義**能現場 demo 的最小可行版（MVP）**，把一條完整流程跑通跑順，再加功能。完整跑通 1 條 > 半殘 5 條。

## 八、決賽展演 ＆ 直式影片致勝建議

決賽 = 影片（3–5 分鐘、直式 720×1280）＋ 現場 6 分鐘實體展演，合計影響「完整性 40 ＋ 展演 30」共 70 分。

**影片（直式、適合手機直拍）**
1. 前 15 秒先講「**誰、什麼痛點**」（用真實人物情境開場，別先講技術）。
2. 中段拍**真人實際使用 Kebbi**的一鏡到底片段（會動、會看、會回應）。
3. 結尾講成效與價值（數據、使用者回饋）。
4. 字幕務必清楚（直式適合大字幕），符合 mp4／≤200MB。

**現場 6 分鐘展演**
- 設計 1 個「**Kebbi 會動/轉頭/比手勢**」的高光時刻，現場最抓眼球。
- 準備**離線備案**：網路/辨識失敗時的 fallback，避免現場翻車（評審看的是穩定的整體成果）。
- 分工：1 人口述故事、1 人操作、1 人盯機器狀態。

**初賽 3 頁文件（教案格式）必含**：科目/領域、教學對象、教學時數、學習目標、預期成果、**對應 108 課綱核心素養**、課程架構圖、**Kebbi 教案設計（功能怎麼用）**、教學成果與回饋、參考資料。➜ 把「用到哪些 Kebbi 功能、解決什麼需求」寫得具體，直接對應「應用性 35」。
