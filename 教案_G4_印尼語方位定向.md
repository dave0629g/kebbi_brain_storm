# 初賽教案草稿 v1｜G4《Tebak Arah：聲源方位 × 印尼語方向詞的真人定向遊戲》

> **草稿 v1**（依建議主攻；待確認後微調）。大會初賽 9 項格式、≤3 頁。評分：應用性 35／創意性 35／挑戰性 30。
> 遊戲邏輯已實作自測：`robot_app` 的 `TebakArahGame`（預設 demo／`--g4t`／`--g4e`），實機 TTS/STT/LLM 已驗證。

## 1. 科目領域
第二外語（印尼語：方位詞）為主，融入資訊科技（Python＋雲端 ASR/翻譯/LLM＋機器人控制）與空間認知。

## 2. 教學對象
高中一年級升二年級（15–16 歲），新住民語文／第二外語選修；程式深度：Python＋串接雲端 API＋機器人動作底盤控制。

## 3. 學習目標
- 能正確使用印尼語方位詞（depan/belakang/kanan/kiri，進階含 serong 斜向 8 向）描述人/物的相對位置。
- 能理解「視角轉換」：同一目標「相對 Kebbi」與「相對自己」方位常相反。
- 體驗「聲源定位」作為方位詞對錯的客觀真值。

## 4. 預期成果
學生用印尼語自報方位，Kebbi **只憑聲源定位**轉頭精準指向發話者、以 DOA 真值判定方位詞對錯並用 LLM 給針對性糾錯；進階玩「裁判賽」（A 描述 B 的方位、Kebbi 核對）、「視角轉換題」與多輪排名。

## 5. 對應課綱核心素養
- **第二外語/新住民語文**：方位詞彙、聽說溝通、跨文化理解。
- **科技 C**：串接 ASR/翻譯/LLM 解決真實語言學習任務。
- **空間認知（數學/地理）**：方位、相對座標、視角轉換。

## 6. 課程架構圖
```
第1節 四向方位詞 → 第2節 聲源定位定向 → 第3節 視角轉換 → 第4節 裁判賽排名+八向延伸
 depan/kanan...    Kebbi 轉頭指人判對錯    相對我 vs 相對Kebbi   round-robin / serong 斜向
```

## 7. Kebbi 教案設計（核心；標出「只有機器人做得到」）
**器材**：1 台 Kebbi（方向性麥克風陣列）、學生圍坐四周、雲端 Azure 印尼語 TTS/STT＋LLM。

- **第 1 節・方位詞**：教 depan/belakang/kanan/kiri；學生圍坐，先校準各人座位方位（Kebbi 用 DOA 記錄）。
- **第 2 節・聲源定位定向**：Kebbi 出題「誰在我的右邊？」→ 學生用印尼語自報 → Kebbi **憑聲源定位轉頭指向**發話者、用 **DOA 真值**判定方位詞對錯 → 答錯用 LLM 給一句印尼語糾錯。〔**平板做不到**：方向性麥克風定位 + 轉頭面向真人〕
- **第 3 節・視角轉換**：要求學生同時說出 B「相對 Kebbi」與「相對自己」的方位（常相反）；兩者都對才得分。〔空間視角的具身教學〕
- **第 4 節・裁判賽＋八向延伸**：A 描述 B 方位、Kebbi 當裁判核對並累積排名；進階引入 serong（斜向）8 向方位詞——並體驗「頭部物理可達性」：正後方頭轉不到，Kebbi 誠實回報「只能面向到最近可達方位」。
- **差異化**：入門→四向；進階→八向＋視角轉換＋裁判賽排名。

**對應評分**：應用性（語言課可直接用、真人互動）／創意性（聲源定位當真值、轉頭指人、視角轉換）／挑戰性（雲端 ASR/LLM 整合＋八向＋馬達可達性判決）。

## 8. 教學成果與回饋
- 形成性：每題「方位詞對／回答者對」即時計分＋ LLM 糾錯；排名表呈現進步。
- 總結性：能否正確用八向方位詞並完成視角轉換題。
- 預期回饋：用真實聲源與轉頭把抽象方位「定錨」到身體與空間，強化記憶與動機。
