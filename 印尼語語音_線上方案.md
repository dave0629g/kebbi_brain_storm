# 印尼語 ASR / TTS 線上方案評估（G4 備援路線）

> 用途：萬一 Kebbi 內建語音**不支援印尼語**，G4 就用 **Unity 經 HTTP 串接外部雲端**做印尼語辨識(ASR)與合成(TTS)。在 SDK 答案回來前先把這條路探好——結論是**完全可行、且競賽期間基本免費**。

---

## 一、結論（先講）
- **首選：Microsoft Azure AI Speech**——印尼語 **STT + TTS 一站到位**、神經語音(Gadis 女聲 / Ardi 男聲)、免費額度大、REST 好接。對 G4 最省事。
- **次選：Google Cloud Speech-to-Text + Text-to-Speech**——印尼語齊全、語音品質高、id-ID 語音選擇更多。
- **想要 LLM 一條龍**：OpenAI（Whisper STT + GPT 出題 + OpenAI TTS），單一金鑰串到底，印尼語可用（TTS 在地感略遜 Azure/Google）。
- **要最自然的印尼語配音**：ElevenLabs（印尼語品質佳，免費 10,000 字/月）。
- **離線備案（會場 Wi-Fi 不穩時）**：**Piper**(印尼語 TTS，輕量可離線) + **Vosk / bookbot 印尼語模型**(離線 ASR)，bundle 進 App 做降級。

> 重點：所有雲端服務都提供 **REST API**，Unity 用 `UnityWebRequest` 即可「錄音→送出→拿文字 / 送文字→拿語音→播放」，**不依賴 Kebbi 內建是否支援印尼語**。

---

## 二、ASR（印尼語 語音→文字）比較

| 服務 | 印尼語 id-ID | 介接 | 免費額度 | 備註 |
|---|---|---|---|---|
| **Azure AI Speech** | ✅ 支援 | REST / WebSocket(即時) | **F0：每月 5 小時免費** | 與 TTS 同一把金鑰，最省事 |
| **Google Cloud STT** | ✅ 支援 | REST / gRPC 串流 | 每月 60 分鐘免費 | 準確度高、口音耐受好 |
| **AWS Transcribe** | ✅ 支援 | REST | 首年每月 60 分鐘 | 綁 AWS 生態 |
| **OpenAI Whisper** | ✅ 支援(WER 微調後可到個位數) | API($0.006/分) 或自架開源 | API 無免費；自架免費 | 可與 GPT 同廠串接；自架可離線(較重) |
| **AssemblyAI** | ◯ 99 語言含印尼語 | REST | 有試用額度 | $0.15/小時起 |
| **Deepgram** | ⚠️ 待確認(Nova 45+ 語言) | REST/串流 | 有試用 | 印尼語支援需確認 |

## 三、TTS（印尼語 文字→語音）比較

| 服務 | 印尼語 id-ID 語音 | 介接 | 免費額度 | 備註 |
|---|---|---|---|---|
| **Azure TTS** | ✅ **id-ID-GadisNeural(女)、id-ID-ArdiNeural(男)** 神經語音 | REST | **每月 50 萬字神經語音免費** | 最推薦，音質自然 |
| **Google Cloud TTS** | ✅ id-ID 多款(WaveNet/Neural2/Chirp3-HD) | REST | 標準音每月 100 萬字、神經音 100 萬字（首份額度） | 語音選擇多 |
| **ElevenLabs** | ✅ 印尼語(多語 v2，可調口音) | REST | 每月 10,000 字 | 最自然、最像真人；量大較貴 |
| **OpenAI TTS** | ✅ 多語可念印尼語 | API | 無免費 | 與 GPT 同廠 |
| **Amazon Polly** | ⚠️ **可能不支援印尼語**(需確認) | REST | 首年每月 500 萬字 | 歷史上 Polly 無 id-ID，採用前務必確認 |

## 四、推薦管線（以 G4 旗艦《Tebak Arah》為例）
```
Kebbi 麥克風收音 ──(Unity 取 mic buffer)──▶ 錄成 wav/pcm
        │
        ▼  UnityWebRequest POST（音訊 bytes）
  Azure STT (id-ID)  ──▶ 印尼語文字（學生說的方位詞 kiri/kanan…）
        │
        ▼  （需要時）呼叫 LLM 出題 / 判錯 / 生成糾錯回饋
   LLM API (Claude/GPT)  ──▶ 回饋文字（印尼語）
        │
        ▼  UnityWebRequest POST（文字）
  Azure TTS (id-ID-GadisNeural)  ──▶ 回傳 mp3/wav
        │
        ▼  Unity 用 Kebbi 喇叭播放
```
> 同理可用於 G4 的另外兩個備案（鏡像體操喊印尼語口令、Pasar 雙機殺價對話）。

## 五、成本：競賽期間基本 $0
- Azure F0：每月 **STT 5 小時 + TTS 神經音 50 萬字**免費；Google：STT 60 分鐘 + TTS 約百萬字免費。
- 一場課堂/demo 的用量遠低於免費額度 → **競賽期間幾乎不花錢**。賽後若要長期部署再評估付費層。

## 六、離線備案（會場網路不穩的保險）
- **TTS 離線**：**Piper** 有印尼語語音(id_ID，輕量、CPU 可跑、樹莓派等級即時)→ 可打包進 App，斷網也能說印尼語。
- **ASR 離線**：Vosk 官方語言清單未含印尼語，但有社群專案 **`bookbot-kids/speech-recognizer-bahasa-indonesian`**(Kaldi/Vosk，支援 Android)可離線辨識印尼語；或自架 Whisper small。
- 策略：**雲端為主、離線為備**——偵測到斷網就切換離線引擎，確保決賽不因 Wi-Fi 翻車（呼應各隊「離線降級備案」原則）。

## 七、Unity 整合與注意事項
- **前置依賴**：需要能在 Unity 取得**原始麥克風音訊**並用喇叭**播放自有音訊**——這正是已送 SDK session 的問題 **E14**；若 Kebbi 開放 mic buffer 與音訊播放，整條雲端管線即成立。
- **介接**：用 `UnityWebRequest` 做 REST（multipart / Base64 音訊）；解析 JSON 取文字；TTS 回傳音訊存暫存檔再播。
- **金鑰安全**：API key **不要硬編進 App**；競賽期可放設定檔或走一個簡單的後端代理轉發，避免外洩。
- **延遲**：雲端來回約 0.5–2 秒，對「輪流問答 / 逐句教學」可接受；錄音片段越短越快。
- **隱私**：學生語音會上傳雲端，課堂使用先告知；離線引擎可在意隱私時改用。
