# 初賽教案草稿 v1｜G5《雙機接力：歷史現場法庭辯論劇場》

> **草稿 v1**（依建議主攻；待確認後微調）。大會初賽 9 項格式、≤3 頁。評分：應用性 35／創意性 35／挑戰性 30。
> 遊戲邏輯已實作自測：`robot_app` 的 `DebateGame`（`--g5`/`--g5t`），多機分散式（G5Director）實機已驗。

## 1. 科目領域
歷史／公民（史料論證、思辨）為主，融入資訊科技（完整系統開發、多機與機器人控制整合）。

## 2. 教學對象
高中三年級升大學（17–18 歲），程式深度：完整系統開發 / 多機與機器人控制整合。

## 3. 學習目標
- 能就歷史爭點（如伽利略宗教審判）建構正／反方論證並引用史料。
- 能評估論證強度、形成判斷（投票）並說明理由。
- 體驗「具身辯論」：空間對峙、轉身點名出證如何強化論證的臨場說服力。

## 4. 預期成果
兩台 Kebbi 分飾對立陣營，分處教室兩端、逐回合**事件接力**辯論、**向中央移動逼近對峙**、關鍵時刻**轉身向學生席點名出證**；學生席可**舉手插話**補充（Kebbi 用姿態確認＋轉頭收音）；結辯依累積投票宣判、勝方舉手致意。

## 5. 對應課綱核心素養
- **歷史/公民**：史料證據的解讀與論證、多元觀點與思辨。
- **A2／C1**：系統思考、道德實踐與公民意識。
- **科技**：完整系統開發、多機協作整合。

## 6. 課程架構圖
```
第1節 爭點與史料 → 第2節 雙機接力辯論 → 第3節 逼近+轉向學生 → 第4節 結辯投票+插話反思
 正/反方論點      事件交棒、指控/攤手    中央對峙、點名出證     累積投票宣判、學生插話改票
```

## 7. Kebbi 教案設計（核心；標出「只有機器人做得到」）
**器材**：2 台 Kebbi（分處兩端）、學生席、雲端中文 TTS/LLM。

- **第 1 節・爭點與史料**：以「地球是否靜止（伽利略案）」為例，學生分組整理正／反方論點與史料。
- **第 2 節・雙機接力辯論**：控方陳述＋伸臂指控 → **事件交棒** → 辯方反駁＋攤手 → 交棒回控方。〔**平板做不到（最強平板免疫）**：兩個實體在空間兩端接力對話、肢體指控/攤手〕
- **第 3 節・逼近與轉向學生**：關鍵爭點兩機**向中央移動逼近對峙**；用**聲源定位轉身**向學生席點名「請這位同學出示證據」。〔**平板做不到**：實體逼近的張力 + 轉身點名〕
- **第 4 節・結辯投票＋學生插話**：逐回合學生/評審投票，結辯依累積票數宣判、勝方舉手；學生席**舉手插話**可改票（Kebbi 用姿態 gate 確認真的舉手才採計）。進階：**分散式多機**（一台中控、一台被控，台詞與動作經 UDP 跨機）。
- **差異化**：入門→照腳本接力；進階→自編論證腳本、整合多機分散式控制。

**對應評分**：應用性（歷史/公民思辨課）／創意性（具身法庭、逼近對峙、轉身點名）／挑戰性（多機接力＋分散式控制＋投票/插話系統）。

## 8. 教學成果與回饋
- 形成性：逐回合投票即時呈現論證強度；插話改票讓學生參與判決。
- 總結性：能否建構有史料支撐的論證並完成一場結構完整的辯論。
- 預期回饋：空間對峙與轉身點名把「論證的說服力」具象化，提升史料思辨的投入度。
